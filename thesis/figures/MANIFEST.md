# Thesis figure manifest

Every figure below is generated by the committed script
[`chaosprobe/scripts/thesis_figures.py`](../../chaosprobe/scripts/thesis_figures.py)
from **archived** run artifacts only — no live cluster access, and every number
printed on a figure is computed from the source data at generation time.

- **Run all commands from `chaosprobe/`** (the data paths below are relative to it).
- Full set: `uv run python scripts/thesis_figures.py --out-dir ../thesis/figures --figure all`
- One figure: append `--figure N`.
- **Generation commit:** the committed PNGs were produced by the same commit
  that introduced this manifest and the script (they are committed together;
  verify with `git log --follow thesis/figures/`), on top of `main` @ `cd7f34a`.
- The ICC bootstrap is seeded (`seed=42` in `chaosprobe.metrics.statistics`),
  so regeneration is deterministic for identical inputs.

## Pending: the 6-strategy node-drain gradient run

At generation time the gradient node-drain run (`results/20260610-172352`) had
only a `partial_summary.json` — no completed `summary.json` — so it is **not**
included. Once it completes **and** passes
`uv run chaosprobe doctor -s results/20260610-172352/summary.json --strict`,
regenerate the H6-dependent figures with the same script:

```bash
uv run python scripts/thesis_figures.py --out-dir ../thesis/figures \
    --figure 2,8,9 --gradient-run results/20260610-172352
```

`fig-09` then gains real y-values for the strategies currently shown as rug
marks, and `fig-02`'s node-drain column counts update automatically.
