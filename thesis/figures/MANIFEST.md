# Thesis figure manifest

Every data figure below is generated by the committed script
[`chaosprobe/scripts/single_study_figures.py`](../../chaosprobe/scripts/single_study_figures.py)
from the **archived** C1/C2/C3 campaign sessions only — no live cluster access.
Each figure reuses the campaign's confirmatory analysis driver
(`c1_h1_trend`, `c3_h2_dns`, `c2_h3_anova`, `v2_h4_frontier`, `scorecard`), so
every number printed on a figure is the same value the results chapter reports.
The workflow schematic (`fig-01-workflow.png`) is reused from `thesis_figures.py`.

- **Run all commands from `chaosprobe/`** (the data paths below are relative to it).
- Full set: `uv run python scripts/single_study_figures.py --out-dir ../thesis/figures --figure all`
- Subset: append `--figure h1,h5` (choices: `workflow,h1,h2,h3,h4,h5`).
- The ICC bootstrap is seeded (`seed=42`), so regeneration is deterministic for
  identical inputs.
